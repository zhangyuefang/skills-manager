# Skills Manager Template\n\n最小可用的 VS Code 扩展模板，功能：\n- 在 Activity Bar（主侧栏）显示图标 “Skills”\n- 点击侧栏图标后在侧边栏显示一个 Webview 管理页（Skills Manager）\n- 提供命令 `skills.openPanel` 可打开为独立的 Webview Panel\n\n如何使用：\n1. 把项目文件保存到本地文件夹（例如 `skills-manager-template`）\n2. 在项目根目录运行：\n   - npm install\n   - npm run compile\n3. 在 VS Code 中按 F5 运行（Run Extension），会打开一个新的 Extension Development Host：\n   - 在主侧栏可以看到 "Skills" 图标，点击打开管理页\n   - 在命令面板（Ctrl+Shift+P / Cmd+Shift+P）运行 `Open Skills Panel` 命令可以打开独立面板\n\n开发提示：\n- 将业务 UI 做成单页应用（React/Vue/Svelte）时，可把打包输出放到 `webview/`，并通过 `webview.asWebviewUri` 引用静态资源。\n- 在扩展端使用 `context.globalState` / `context.secrets` 做本地持久化或 API key 存储。\n- 如果想扩展为支持 Claude/OpenAI provider，可以在 src 下添加 providers/ 抽象，并通过 postMessage 在 webview 与扩展端通信。\n
